# 提交工单

提交工单页面是用户发起业务申请的入口。系统以工单模板为单位，按分类组织展示，用户选择对应模板后填写表单即可完成工单创建。

<img src="/images/ticket/ticket-start.png" style="width: 100%; border-radius: 8px; margin: 16px 0;" alt="提交工单" />

## 1、页面布局

页面采用左右分栏布局：

- **左侧边栏**：模板分类导航，支持快速筛选
- **右侧内容区**：模板卡片列表，支持搜索与刷新

## 2、分类导航

左侧边栏提供三种分类入口：

| 分类 | 说明 |
|------|------|
| 我的收藏 | 展示用户收藏的模板，快速启动常用工单 |
| 全部工单 | 展示所有可用模板，按分类分组显示 |
| 自定义分类 | 管理员配置的业务分类，如运维、研发等 |

## 3、模板卡片

每张模板卡片展示以下信息：

- **图标**：模板配置的业务图标
- **名称**：模板名称
- **描述**：模板用途说明
- **收藏按钮**：点击星标可收藏 / 取消收藏该模板

点击卡片或"立即创建"即可打开工单填写弹窗。

## 4、创建工单

点击模板卡片后，弹出工单创建对话框：

- 系统自动加载该模板配置的动态表单
- 填写完成后点击**提交工单**，工单进入流程引擎开始流转
- 提交成功后可在[工单列表](./ticket.md)的**我的工单**标签页中跟踪进度

> [!TIP]
> 对于常用的工单模板，建议点击星标收藏，下次可通过**我的收藏**快速找到。
